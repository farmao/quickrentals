About the Game

Azul is a visually stunning, award-winning board game where players compete to decorate a palace wall using beautifully crafted ceramic tiles. Each turn, you’ll draft tiles and strategically place them to score points based on patterns and positioning. It’s quick to learn but offers deep strategy, making it great for casual game nights or more competitive play. Every round is a balance of timing, planning, and a little bit of luck.

Game Features:

  • 2-4 Players
  • 30-45 Minutes Playing Time
  • Ages 8+
  • Beautiful artwork and high-quality components
  • Easy to learn, challenging to master
